## What Is Now Frozen

The bounded post-article plugin lane now has one explicit admissibility
contract that:

- keeps candidate sets closed-world and explicitly enumerated
- keeps equivalent-choice classes explicit and auditable
- binds admissibility to named route policy and world-mount posture
- binds plugin version constraints, trust posture, and publication posture
- compiles capability, network, artifact, timeout, memory, provenance, and
  replay posture into runtime-owned envelopes
- keeps denied, suppressed, and quarantined outcomes typed instead of implicit

## What This Still Does Not Claim

This tranche does not claim:

- weighted plugin control
- plugin publication rights
- served or public plugin universality
- arbitrary public software execution

The remaining frontier stays on later conformance, composition, controller,
authority, and platform issues.
